## Aplicación para Predicción con Streamlit

Este proyecto implementa una aplicación interactiva utilizando **Streamlit** que permite a los usuarios cargar un archivo **Excel**, realizar predicciones basadas en un modelo previamente entrenado, y descargar el archivo con las predicciones agregadas.

### Funcionalidades:
- **Carga de Archivos Excel**: Los usuarios pueden cargar un archivo Excel con los datos a ser procesados.
- **Predicción Automatizada**: Utilizando un modelo de predicción entrenado, la aplicación genera predicciones para los datos cargados.
- **Exportación del Archivo**: Una vez que se generan las predicciones, el archivo con las predicciones añadidas puede ser descargado por el usuario en formato Excel o CSV.
